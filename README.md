# Billy - AI-Powered Crypto Bill Payment Assistant

An intelligent assistant that helps you pay and clear bills before they're overdue using cryptocurrency payments.

## Project Structure

- `backend/` - API server for bill processing and AI integration
- `contracts/` - Smart contracts for payment logic
- `mobile/` - Flutter mobile application with BLoC architecture

## Tech Stack

### Frontend (Mobile)
- Flutter with BLoC state management
- Web3dart for blockchain interactions
- Secure storage for wallet management

### Backend
- Node.js/TypeScript API server
- AI integration for bill parsing
- Payment gateway services

### Blockchain
- Ethereum/Polygon for payments
- Stablecoin support (USDC, USDT, DAI)
- Smart contracts for automated payments
