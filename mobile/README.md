# Billy Mobile App

Flutter application with BLoC architecture for managing bill payments with crypto.

## Setup

```bash
flutter pub get
flutter run
```

## Features

- Bill management (add, view, update, delete)
- AI-powered bill parsing from images
- Crypto wallet integration
- Payment processing with Web3
- Payment history tracking

## Architecture

- BLoC pattern for state management
- Repository pattern for data layer
- Service layer for API and Web3 interactions
